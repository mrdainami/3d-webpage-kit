# Brand — fill this in (Claude reads it before every build)

This file tells Claude who the brand is, so copy and design come out in
character instead of generic. Edit it, or run `/set-brand` to have Claude
interview you and write it for you.

## Identity
- **Name:** Your Brand
- **What it sells:** (one line)
- **Audience:** (who it is for, in plain words)

## Voice
- **Feel:** confident, concrete, calm. Not hypey.
- **Sentences:** short. One idea each.
- **Do:** lead with the benefit, name the outcome, use real specifics.
- **Do not:** filler adjectives, buzzwords, exclamation spam, em dashes.

## Look
- **Mood:** premium, dark, editorial. (Change in `tokens.css`.)
- **Accent color:** set `--accent` in `tokens.css`. One accent only.
- **Type:** display + body set in `tokens.css`.

## Reference copy (tone sample)
> We did not build another blender. We engineered a force of nature that
> reduces anything to nothing, in silence.

Replace with 2 to 3 lines that sound like your brand. Claude matches this tone.
